# Look-Alkemy v2

A next-generation React Native/Expo mobile application.

## 🚀 Setup

```bash
# Install dependencies
npm install

# Start Expo dev server
npm start

# Run on Android (requires Expo Go app)
npm run android

# Run on iOS (requires Expo Go app)
npm run ios

# Run on web
npm run web
```

## 📱 Development

This project uses Expo for cross-platform development. To test on your device:

1. Install **Expo Go** from App Store (iOS) or Play Store (Android)
2. Run `npm start`
3. Scan the QR code with Expo Go

## 🏗️ Architecture

- **Framework:** React Native with Expo
- **Language:** TypeScript
- **Routing:** Expo Router (file-based routing)
- **Styling:** NativeWind / Tailwind CSS (optional)

## 📋 Requirements

- Node.js 18+
- npm or yarn
- Expo Go app (for mobile testing)
